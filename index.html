MINHAS ESPECIALIDADES.

JAVA SCRIPT

JavaScript é uma linguagem de programação voltada para a criação de interatividade em sites. Com ele, é possível manipular elementos da página, responder a ações do usuário e criar experiências dinâmicas no navegador.

NODE. JS

Node.js é uma plataforma que permite executar JavaScript no lado do servidor. Com ela, é possível criar APIs, servidores web e aplicações completas usando uma única linguagem em todo o projeto

C#

C# é uma linguagem de programação moderna e versátil desenvolvida pela Microsoft. Muito usada para criar aplicações desktop, web, jogos com Unity e sistemas corporativos robustos.

MEU PORTIFÓLIO.

VERDAZZO

VERDAZZO Burger - Página de Cardápio Interativa.

Este projeto simula uma experiência real de pedido online e demonstra habilidades em front-end responsivo, interatividade com JavaScript, e design centrado no usuário.

  • HTML5 para estruturação da página.
  • Tailwind CSS para estilização moderna e responsiva.
  • JavaScript puro para interatividade e manipulação do carrinho.
  • Toastify.js para notificações visuais ao adicionar produtos.
  • Font Awesome para uso de ícones nos botões e interface.
  • Layout responsivo e moderno com uso de Tailwind CSS.
  • Exibição de produtos (hambúrgueres e bebidas) com imagens, descrições e preços.
  • Botões de "Adicionar ao carrinho", que permitem ao usuário montar seu pedido.
  • Modal de carrinho de compras com total do pedido e campo para endereço.
  • Integração com Toastify.js para notificações visuais.
  • Ícones interativos com Font Awesome.
